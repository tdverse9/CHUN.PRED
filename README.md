# Customer Churn Prediction using Artificial Neural Network (ANN)

This project predicts whether a customer is likely to churn a service based on historical data, using a deep learning model built with ANN.

Problem Statement
Customer churn is a critical issue for businesses. Early identification of at-risk customers allows companies to take proactive retention measures. This model helps predict churn with high accuracy using customer data.
